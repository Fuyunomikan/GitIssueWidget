## 実装方針メモ（2026-05-05 確定）

- **期限フィールドはスコープ外**: GitHub Issues にネイティブの DueDate がなく、Projects v2 の正規実装は GraphQL 必須で工数が大きいため、本イテレーションでは期限関連を全て削除
- **要件 (1) のプルダウン UI**: Glance 単体ではドロップダウン UI が描けないため、ウィジェットヘッダの「⋮」アイコンから Dialog テーマの `WidgetQuickEditActivity` を起動する案で実装
- **要件 (3) のスコープ**: アプリ全体設定として扱う（`PreferenceStore` に格納、全ウィジェットに一律適用）
- **要件 (2) のスコープ**: リポジトリ・タイトル・ラベルのみ。ラベルは `GET /repos/{owner}/{repo}/labels` で候補取得後に複数選択
- **実装順**: (2) → (3) → (1) — 後の項目が前の追加 API/プロパティを再利用する関係
